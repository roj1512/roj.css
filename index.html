Roj.css

My first CSS library.

Some of its cool features are:

  • Simple & small
  • Built-in dark & light mode
  • Ability to use user-defined themes
  • Built-in RTL support (preview)
Note: I made this for my own usage, I’m not sure if it works for you.

Using It Yourself

  1. Choose a variant.

    • roj: LTR, decides on choosing dark mode
    • roj-dark: LTR, dark mode only
    • roj-light: LTR, light mode only
    • roj-rtl: RTL, decides on choosing dark mode
    • roj-rtl-dark: RTL, dark mode only
    • roj-rtl-light: RTL, light mode only

  2. Choose a style.

    • .css: Takes longer to load, easier to read
    • .min.css: Takes shorter to load, harder to read

  3. Import it.

    <link rel="stylesheet" href="https://roj1512.github.io/roj.css/dist/roj.css" />
Note: the statement above is importing roj.css. You might modify it to your needs.
  1. Write HTML and it’ll get styled. :)

Using Your Colors

Roj.css depends only on 5 different colors, and you can change them anytime: 

:root {
  --dark: #222222;
  --dark-alt: #333333;
  --light: #eeeeee;
  --light-alt: #cccccc;
  --color: #1379ce;
}

RTL Support

Roj.css supports RTL. Just import an RTL variant, as mentioned in above. And set the dir attribute of the html tag to rtl: 

<html dir="rtl">
